## Deploy
```bash
git init && git add . && git commit -m "VOLTLOT v5 — live"
git remote add origin https://github.com/YOUR_USERNAME/voltlot.git
git push -u origin main
```
Settings → Pages → main → Custom domain: voltlot.ai

## Quick Deploy
- **Netlify**: Drag folder → app.netlify.com/drop
- **Vercel**: npx vercel --prod
- **Cloudflare Pages**: Connect repo, no build step
